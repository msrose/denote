Module: coerce

(require("coerce"))(value, Type, fulfill, reject) → {undefined}

Implements the promise resolution procedure for the given type. The promise adopts the state of its resolve value.
Parameters:
Name Type Description
value any The value that the promise is resolved with
Type function The constructor function of the promise type to coerce. Just pass noop (i.e. function() {}) if you don't have a constructor for your promises.
fulfill function Callback called with the final resolution if coercion ends with fulfillment
reject function Callback called with a rejection reason if coercion ends with rejection
